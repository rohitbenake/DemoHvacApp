# DemoHvacApp

DemoHvacApp is Android Automotive app for showing temparature.
The application communicates with Vehicle HAL using Car service and displays the temperature changes.
  
Android App -->Car Service -->VHAL
                                  | |
                                  | CAN Network | <--> CAN BUS
                                  | |
Android App <--Car Service <--VHAL
